# Wonder Hub 三层隐性漏斗设计 v3.0
> 核心约束：Derek 为 AIA 直属代理人，公开层须低调，以私域为核心变现场
> 修订日期：2026-04-17

---

## 一、核心原则修订

> [!IMPORTANT]
> **AIA 直属代理人的公开营销限制（关键认知）**
> 
> AIA 直属代理人受以下双重约束：
> 1. **保险监管局（IA）**：所有招揽行为须在持牌环境下进行
> 2. **AIA 内部合规**：所有以 AIA 代理人身份进行的公开营销材料，须经 AIA Compliance 预先审批
>
> 因此，公开层**绝对不能**：
> - 提及 AIA、提及具体产品
> - 做保险公司比较
> - 以"保险顾问"身份明显出现
> - 直接推介任何受规管的保险产品

---

## 二、三层分离架构

第一层：公开层（Wonder Hub 公开网站 + SME系统）  
身份：纯粹的内容平台 / 中小企工具  
不见：Derek 姓名 / 代理人身份 / 保险公司名称

↓（过滤：只有真正感兴趣的人进入）

第二层：私域层（邮件列表 / WhatsApp）  
身份：开始建立个人信任，渐进式披露专业背景  
出现：Derek 声音（但仍不主打 AIA）

↓（过滤：只有有需求的人进入）

第三层：咨询层（1对1面谈 / 视频通话）  
身份：Derek 以持牌AIA顾问身份正式出现  
使用：AIA 预审批材料，完整合规流程

## 三、第一层：公开层设计

### Wonder Hub 网站公开呈现策略

品牌定位：服务香港中小企及有投资意识人士的知识平台

| 公开内容 | 描述 |
|---------|------|
| SME 管理系统 | 本身价值，不附带任何保险推广 |
| 投资理念专栏 | Wonder Portfolio 月度追踪，注明仅供教育参考 |
| MPF 知识库 | 教育性内容（MPF 如何运作、整合须知） |
| 香港家庭财务规划指南 | 通用内容，无产品推荐 |
| 财务健康问卷 | CTA核心入口 → 引入私域 |

### 第一层 CTA 设计（引流钩子）

不要直接问"需要买保险吗"，而是提供价值换取联系方式：

钩子A：免费工具  
"下载：2026 香港中小企主必备合规清单（PDF）"  
→ 留邮件 → 进入邮件序列

钩子B：问卷诊断  
"5分钟财务健康自测：你的家庭保障缺口有多大？"  
→ 完成后"查看分析，需加入我们的专属社群"  
→ 进入 WhatsApp / 邮件私域

钩子C：SME系统内置  
"一键生成公司年度合规检查报告"  
→ 报告末尾："如需专业人士协助，欢迎加入我们的顾问服务群"

### SME 系统合规提醒的措辞策略

合规且引流的提醒设计示例：

📋 雇主责任险提醒  
您的雇主责任险将于 [日期] 到期。  
根据香港《雇员补偿条例》，这是法定要求。  
[了解详情] → 链接到 Wonder Hub 知识库文章  ← 引流网站  
[加入顾问群] → 引入 WhatsApp 私域            ← 进私域  
（不出现：AIA、任何保险公司名称）

## 四、第二层：私域层设计

### 私域工具选择

| 工具 | 适合场景 | 优先级 |
|-----|---------|-------|
| WhatsApp Business + 广播列表 | 高黏性、私人感、香港用户习惯 | 首选 |
| 邮件订阅（Mailchimp/Beehiiv） | 内容较长、适合存档 | 辅助 |
| WhatsApp 小群（10-20人） | 有话题氛围的小圈子 | 进阶 |

### 私域层内容策略（渐进式建立信任）

加入后第1-2周（纯价值输出，不露身份）：
- 每周分享1条实用财务知识（MPF 小知识、税务提醒等）
- SME 合规贴士
- Wonder Portfolio 简短更新

第3-4周（开始建立个人连接）：
- "我最近在思考..." 个人观点式内容
- 开始署名"Derek"（但不提AIA）
- "如果你对香港财务规划有任何疑问，欢迎私聊"

第5周以后（自然展示专业身份）：
- 当有人主动问起才透露持牌财务规划顾问身份
- 不主动说AIA，等对方问
- 对话自然流向"预约一次免费咨询吧"

## 五、MPF 的特殊地位（最佳公开话题）

MPF 是整个模式中最适合公开讨论的话题：
1. MPF 归 MPFA 监管，与 AIA 直属无关
2. MPF 是所有香港雇主的强制性义务
3. 教育内容空间大（供款、整合、基金选择）
4. SME 系统用户天天都在算 MPF

MPF 内容策略：

公开层：
- MPF 整合教育文章（高搜索量内容）
- 老板必看：新员工MPF注册常见错误
- Wonder Hub MPF 扣款计算器（工具引流）

进私域：
- 免费 MPF 健康检查 → 加入 MPF 优化小组
- WhatsApp 内定期发送 MPF 基金表现简报

咨询层：
- MPF 整合方案（MPFA持牌，独立于AIA）

## 七、一句话总结

Wonder Hub 的公开层是一个纯粹的知识与工具平台，与保险业务完全隔离；  
私域层是 Derek 建立信任关系的花园，慢慢让对方认识真实的专业价值；  
咨询层才是唯一发生正式持牌业务的地方。

低调不是弱点，低调是护城河。  
价值在前，身份在后，合规始终。
