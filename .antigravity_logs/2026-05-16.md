## 新增功能：小红书图文卡片生成器 (XHS Creator)
- **目标**：在 Wonder Hub 内实现长文（Markdown）转小红书 3:4 图文卡片的功能。
- **改动位置**：`apps/wonder-hub`
- **主要文件**：
  - `src/lib/xhs-parser.ts`：增加 Markdown 解析与分页逻辑（支持 `---` 强制分页，或根据 `## ` H2 标签自动分页）。
  - `src/components/xhs-creator/XhsTheme.ts`：根据 Frontend Design 规范定义了三个预设主题（杂志编辑风、极简自然风、粗野工业风）。
  - `src/components/xhs-creator/XhsCard.tsx`：单张 1080x1440 比例的图片渲染组件，内部使用 `react-markdown` 渲染。
  - `src/app/xhs-creator/page.tsx`：隐式入口（无 Nav 跳转），提供左中右三栏结构（设置、预览、编辑），并且默认底部署名 "Wonder Wisdom 出品"。
- **新依赖**：安装了 `react-markdown`（Markdown 渲染）和 `html-to-image`（HTML 转高清图片）。
- **更新**：按照用户要求，废弃了原有的 3 套自定义主题，改为 **完全复刻对标网站的 6 套原生主题**（杂志单页、骑士红黑、暖金晚雪、柔和科技、科学期刊、空山新雨）。同时在左侧边栏新增了与原站一模一样的控制选项：字体选择（阿里普惠、仓耳今楷、霞鹜文楷、系统字体）、字体大小滑动条（支持字号动态缩放）、以及背景杂点开关。
- **UI 优化与功能增强 (Wonder Hub XHS)**：
  - **[Status]**: 已完成
  - **[Change Log]**: 修改 `apps/wonder-hub/src/app/xhs-creator/page.tsx` 增加了右侧 Markdown 工具栏，包含加粗、引用、列表等按钮；新增「第一张使用封面排版」复选框控制封面状态；在左侧增加「导入来源」UI，支持 URL 输入和本地 MD 文件上传。新增 `apps/wonder-hub/src/app/api/xhs-ai/route.ts` 作为后端接口，结合 DeepSeek 提取网页并自动生成小红书文案。修复了热更时遗漏 State 导致的报错。
