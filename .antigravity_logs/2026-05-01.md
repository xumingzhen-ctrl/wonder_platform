
## [已完成] i18n 多语言支持系统

**[Status]** 已完成

**[Change Log]**
- 新建 `apps/fis-hub/src/i18n/translations.js` — 完整中英双语词典（compliance/header/footer/role/sidebar/portfolio/lab/common 共8个命名空间）
- 新建 `apps/fis-hub/src/i18n/LangContext.jsx` — React Context + useLang hook + localStorage持久化（默认zh）
- `apps/fis-hub/src/main.jsx` — 用 `<LangProvider>` 包裹 App 根节点
- `apps/fis-hub/src/App.jsx` — 引入 useLang，更新 ComplianceGateModal（传 t prop）、Header（语言切换按钮 + 所有标签）、Footer、Undo 对话框、ROLE_META
- `apps/fis-hub/src/components/Sidebar.jsx` — 引入 useLang，所有导航/标签/操作按钮均通过 t() 输出
- `apps/fis-hub/src/components/PortfolioView.jsx` — 引入 useLang，翻译所有功能按钮/Tab/统计卡片/图表说明
- `apps/fis-hub/src/components/StrategyLabView.jsx` — 引入 useLang，翻译标题/子标题/沙盒标签

**[Logic Reason]**
用户要求全平台中英文切换，默认中文，按钮置于右上角 Header。采用轻量 React Context 方案，无新依赖，localStorage 持久化语言偏好。
