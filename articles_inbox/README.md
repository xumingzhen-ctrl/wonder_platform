# 📥 文章投稿文件夹

把你的 Word 文档（.docx）拖放到这个文件夹中，然后运行转换脚本。

## 使用方法

```bash
# 在 Wonder_Platform 目录下运行：
python3 convert_articles.py
```

## 分类规则（文件命名约定）

在文件名中加入分类前缀，脚本会自动识别：

| 文件名前缀 | 识别为 | 示例 |
|---|---|---|
| `保险_` 或 `ins_` | 保险理念 | `保险_复利的力量.docx` |
| `投资_` 或 `inv_` | 投资理念 | `投资_长期主义.docx` |
| `通讯_` 或 `newsletter_` | 客户通讯 | `通讯_2026Q1季报.docx` |
| 无前缀 | 投资理念（默认） | `资产配置报告.docx` |

## 转换后输出位置

`apps/wonder-hub/src/content/blog/` 目录下，可直接预览。
